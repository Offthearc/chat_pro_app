# chat_pro_app

A real-time chat platform where users register instantly, join AI-article-linked discussion rooms, and message each other directly.

## Prerequisites

- Node.js >= 20
- npm >= 9

## Setup

```bash
git clone git@github.com:Offthearc/chat_pro_app.git
cd chat_pro_app
npm install
```

## Run dev server

```bash
npm run dev
```

Open http://localhost:5173

## Run tests

```bash
npm test
```

## Lint

```bash
npm run lint
```

## Build for production

```bash
npm run build
```
